# 📚 데이터 수집 프로그램 사용법 가이드

안녕하세요! 파이썬과 데이터베이스를 처음 공부하시는 분들을 위해 만든 프로그램입니다.

## 🎯 이 프로그램으로 배울 수 있는 것들

### 1. 파이썬 기초
- 함수 정의와 호출
- 클래스와 객체 지향 프로그래밍
- 예외 처리 (try-except)
- 파일 입출력
- 사용자 입력 받기

### 2. 데이터베이스 기초
- SQLite 데이터베이스 사용법
- SQL 명령어 (CREATE, INSERT, SELECT, UPDATE, DELETE)
- 데이터베이스 연결과 관리
- 데이터 타입 이해

### 3. 실무 프로그래밍
- 메뉴 기반 프로그램 만들기
- 데이터 검증과 오류 처리
- 사용자 친화적인 인터페이스

## 🚀 시작하기

### 1단계: 학습 예제 실행하기
```bash
python learning_examples.py
```
이 명령어를 실행하면 파이썬과 데이터베이스의 기본 개념을 단계별로 학습할 수 있습니다.

### 2단계: 데이터베이스 초기 설정
```bash
python database_setup.py
```
데이터베이스와 테이블을 생성합니다.

### 3단계: 메인 프로그램 실행
```bash
python data_collector.py
```
실제 데이터 수집 프로그램을 실행합니다.

## 📖 프로그램 구조 설명

### 파일별 역할

1. **`learning_examples.py`** - 학습용 예제
   - SQLite 기본 사용법
   - 오류 처리 방법
   - 데이터 타입 이해
   - 파이썬과 데이터베이스 통합

2. **`database_setup.py`** - 데이터베이스 설정
   - 데이터베이스 파일 생성
   - 테이블 구조 정의
   - 초기 설정 확인

3. **`data_collector.py`** - 메인 프로그램
   - 사용자 정보 입력
   - 데이터 저장/조회/삭제
   - 메뉴 기반 인터페이스

4. **`data.db`** - 데이터베이스 파일
   - 실제 데이터가 저장되는 파일
   - 프로그램 실행 시 자동 생성

## 🔧 주요 기능 설명

### 1. 새로운 사용자 추가

## 🐛 자주 발생하는 오류와 해결법

### 1. "ModuleNotFoundError: No module named 'sqlite3'"
- 파이썬이 제대로 설치되지 않았을 가능성
- 파이썬 재설치 필요

### 2. "Permission denied"
- 파일 권한 문제
- 관리자 권한으로 실행하거나 파일 위치 변경

### 3. "database is locked"
- 다른 프로그램이 데이터베이스를 사용 중
- 프로그램을 모두 종료하고 다시 실행
